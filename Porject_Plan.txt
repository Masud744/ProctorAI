Project Title: AI Classroom & Exam Monitoring System

Project Overview:
This project is an AI-based Classroom and Online Exam Monitoring System that uses Computer Vision, Machine Learning, and Web Technologies to monitor students in real-time through a webcam during online classes and exams. The system analyzes student behavior such as attention level, head movement, eye gaze direction, sleep detection, phone usage, multiple person detection, and suspicious activities. Based on these observations, the system calculates an Attention Score during learning sessions and a Suspicious Activity Score during exams.

The system operates in two main modes:

1. Learning Mode (Classroom Monitoring Mode)
2. Exam Mode (AI Proctoring Mode)

In Learning Mode, the system monitors whether a student is paying attention to the screen, sleeping, looking away frequently, or leaving the seat. It calculates an attention score over time and generates analytics such as focus graphs, attendance, and student performance reports.

In Exam Mode, the system acts as an AI proctoring system. It detects cheating-related behaviors such as looking left/right repeatedly, using a mobile phone, multiple faces appearing in the camera, talking during the exam, or leaving the seat. The system logs suspicious activities, captures screenshots, and calculates a suspicious activity score. If the suspicious score crosses a threshold, the system generates alerts.

System Objectives:

* Monitor student attention during online learning
* Automatically track attendance using face detection
* Detect sleep, distraction, and absence
* Detect cheating behaviors during online exams
* Generate attention and suspicious activity scores
* Store logs and analytics in a database
* Provide a teacher/admin dashboard for monitoring
* Generate reports and analytics graphs
* Build a real-time AI monitoring system

Core Technologies:

* Computer Vision (OpenCV, MediaPipe)
* Machine Learning / Deep Learning
* Object Detection (YOLO for phone detection)
* Backend API (FastAPI or Flask)
* Frontend Dashboard (React)
* Database (MongoDB or MySQL)
* Real-time video processing
* Cloud Deployment (Docker, AWS/Render/VPS)

Main Computer Vision Modules:

1. Face Detection
2. Face Recognition (optional for attendance)
3. Eye Gaze Detection
4. Head Pose Estimation
5. Eye Closure / Sleep Detection
6. Mouth Detection (Talking Detection)
7. Phone Detection (Object Detection Model)
8. Multiple Face Detection
9. Person Absence Detection

Attention Score Logic (Learning Mode):
The system calculates attention score based on:

* Looking at screen
* Eyes open
* Head facing forward
* Face present in frame
* Not using phone
* Not sleeping

Suspicious Activity Score Logic (Exam Mode):
The system increases suspicious score when:

* Looking left/right frequently
* Phone detected
* Multiple faces detected
* Talking detected
* Leaving seat
* Face not visible for long time

System Architecture Flow:
Webcam → Computer Vision Models → Behavior Detection → Scoring Engine → Backend API → Database → Dashboard → Reports & Alerts

Database Will Store:

Dashboard Features:

Expected Outcome:
This project will be a full-stack AI system combining Computer Vision, Machine Learning, Backend Development, Database Management, and Frontend Dashboard to build a real-time intelligent student monitoring and exam proctoring system. The system can be used in online learning platforms, universities, training centers, and remote examination systems.
